<h1 align="center">Vue SSR Boilerplate</h1>

## Usage

Install dependencies:

```
yarn
```

Start dev server:

```bash
yarn serve
```

Build for production:

```bash
yarn build
```

Start production server:

```bash
yarn start
```

## Docker

Build & run

```bash
docker-compose up --build -d
```
